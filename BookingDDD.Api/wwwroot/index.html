BookingDDD CQRS API example

Open the browser console and use bookingApi.

const booking = await bookingApi.book(
    "2026-06-15T10:00:00",
    "2026-06-15T11:00:00");

await bookingApi.cancel(booking.id);

await bookingApi.bookings();
await bookingApi.calendar("2026-06-15");
await bookingApi.availableSlots("2026-06-15");
